# FitNit

[FitNit](https://fitnit.netlify.app/) is a fitness app written in Typescript. It is aimed at creating and editing workouts and to efficently make workout routines and stick to them.

## Development

When editing FitNit's frontend module there are 3 components required to run.

### Dev Server
```bash
npm run dev
```

### Typescript Server 
```bash
npm run tsc
```

### Tailwind CSS Server
```bash
npm run tailwind
```

Other useful tools are a command for a local server and a python script to check the size of the source code folder

### Local Test Server
```bash
cd ./frontend && npm run server
```

### Python Size Checker
```bash
cd ./frontend && npm run size
```
